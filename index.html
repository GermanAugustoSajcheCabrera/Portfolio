Projects

Project 1

Overview:

This project focuses on automating critical hospital systems in the US and Guatemala, leveraging Python and Robocorp's RPA tools to streamline operations and enhance efficiency. The solution includes automating data workflows, integrating hospital systems with SAP and SAC, and managing the entire project lifecycle through ETL processes for a US-based company. Additionally, the project encompasses frontend development using React and Angular, and API services integration, enhancing user experience and backend functionality.

Technologies Used:

  • RPA and Automation: Developed automation scripts using Python with Robocorp, applying math and data libraries for data processing and scraping. Automated hospital systems by integrating SAP and SAC for enhanced data management.
  • Data Handling & Versioning: Employed Git and Azure for handling version control and collaborative development across distributed teams.
  • Project Management: Managed ETL projects for a US-based company, coordinating efforts to integrate data across platforms.
  • API Integration: Consumed services using REST & SOAP APIs, ensuring seamless integration with external systems for data exchange.
  • Web Development: Developed responsive and interactive web applications using Angular and Node.js, connecting them with Python-based APIs for backend processing.
  • UI/UX Development: Created components and forms using React (JSX) to build intuitive, user-friendly interfaces for web applications.
  • Database Management: Worked with SQL databases to design and manage stored procedures, views, and application connections, ensuring efficient data storage and retrieval.

Impact:

This project resulted in the successful automation of healthcare processes, improving operational efficiency and reducing manual intervention. The web solutions provided intuitive interfaces and smooth backend integration, while the ETL processes ensured accurate data flow and analysis. The combination of these efforts enhanced overall system performance and user experience in both healthcare and business environments.

Project 2

Overview:

This project revolves around automating business processes, performing data manipulation using advanced machine learning, AI, and OCR techniques, and integrating interactive automations with chatbot functionality. As an RPA Senior Developer and QA, the primary focus was to analyze processes, gather requirements, and create efficient, automated workflows. The project also involved implementing DevOps practices to optimize infrastructure, managing cloud services, and creating interactive dashboards for real-time data monitoring.

Technologies Used:

  • RPA and Automation: Automated business processes by analyzing workflows and implementing intelligent automations, which were integrated with chatbots for real-time user interactions.
  • Data Manipulation and AI: Leveraged machine learning, OCR, and AI techniques to manipulate data, extract information from unstructured sources, and drive intelligent decision-making within automated systems.
  • DevOps Implementation: Implemented DevOps practices, managing infrastructure, tools, and processes for continuous integration and deployment (CI/CD), ensuring high availability and scalability of services.
  • API Integration: Consumed services using REST & SOAP APIs to ensure seamless communication between different systems and automate data flows between external and internal services.
  • Database Optimization: Managed and optimized stored procedures in SQL databases, enhancing the efficiency and performance of data operations.
  • Cloud Services Management: Managed cloud services and infrastructure to ensure optimal performance, while also utilizing cloud platforms to host interactive dashboards that provided real-time data insights and monitoring.

Impact:

The project resulted in streamlined and highly efficient automated processes, enhanced data manipulation through AI and machine learning, and an improved user experience with chatbot-enabled interactions. The implementation of DevOps practices ensured smoother workflows and increased infrastructure reliability, while the optimized stored procedures and cloud-based dashboards improved data accessibility and performance monitoring. This project enhanced overall business operations, leading to improved decision-making and reduced manual effort.
